About Demograph

Demograph is Cheryl Lewman's Capstone Project, a culmination of twelve weeks of intense code school bootcamp at PDX Code Guild. It was intended to show a visual of the disparity between education and income. The United States provides a comprehensive data set called the American Community Survey, ACS. In addition to showing education, gender, and income, this project will eventually include many more data sets from the ACS.

This project uses Django, which connects the database, SQLite, with Python and the frontend. The ACS API was used to populate the database. Plot.ly for Python was used to display the graphs, using PANDAS and Numpy. It also makes use of the typical frontend tools such as: JavaScript, CSS, HTML5, and Materialize.
